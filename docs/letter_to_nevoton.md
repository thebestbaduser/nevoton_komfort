**Тема:** Интеграция KOMFORT-WF с Home Assistant

Здравствуйте!

Я владелец контроллера **NEVOTON KOMFORT-WF** (прошивка 3.5) и пользователь **Home Assistant**. На основе вашего **NEVOTON REST API v1.0** я разработал открытую интеграцию для локального управления сауной из умного дома.

Интеграция уже работает у меня в эксплуатации и поддерживает:
- терморегулятор (включение нагрева, целевая и текущая температура);
- освещение с диммером;
- переключатели (питание, вентилятор, парогенератор, отложенный старт);
- сенсоры температуры, влажности, статуса и таймеров.

Репозиторий: https://github.com/thebestbaduser/nevoton_komfort  
Установка — через HACS, без облака: контроллер опрашивается по локальной сети.
